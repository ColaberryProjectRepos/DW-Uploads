Remote login to the Colaberry Virtual machine: (Step -2)

During your scheduled time go ahead and remote login to Colaberry hosted virtual machine. Below are the steps to follow:

1.       Go to windows search button and search for “Remote Desktop Connection”.

2.       Open the app

 

 

3.       Computer name 104.210.209.47 , user name: Colaberry1

4.       For password reach out to the program director

5.       After entering correct password, you will be able to remotely login to the colaberry VM

 

Now you need to copy the data files and ETL solution in Colaberry Server.

1.       Go to windows file explorer “F” Drive, you will see DW&ETLFileStorage (For the data files) and ETLSolutions (For the etl solutions)

2.       Now create your folder in the above-mentioned folders

a.       Naming convention should be “YourFirstName_YourLastName_YourProjectName” for each of the folders

b.       See the examples exists in the folder (Ex: Moses_Wemegah-NewYorkOilAndGas, Moses_Wemegah-NewYorkOilAndGas_Solution)

3.       Copy your data files and solutions in the folders you just created
