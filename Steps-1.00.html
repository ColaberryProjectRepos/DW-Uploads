Overview

Congratulations on completing your project, now it is time to deploy your project in Colaberry Server environment. This document will guide you on how to connect to the Colaberry Virtual machine and the action items you must complete to be able to deploy your project.

Deployment guidelines:

1.       Schedule your deployment window

2.       Remote login to the Colaberry Virtual machine (Make sure to request password in basecamp private ping before your deployment window starts)

a.       Create folders in "F" drive for data files and ETL Solution

3.       Deploying the project (You will access this server's SSMS & SSIS)

a.       Create database (Staging, ODS, DW as directed in your project requirements)

b.       Populate your database tables

c.       Create folder in integration services catalog

d.       Deploy your packages

4.       Scheduling Packages
